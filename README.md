# Hate Speech Detection Using Machine Learning

This project focuses on detecting hate speech in Twitter data using machine learning techniques.

## Repository Link
[Hate Speech Detection Using LR](https://github.com/liza151/hate-speech-detection-using-LR.git)

## Project Structure
- `Hate speech detection on twitter data_live.ipynb`: Jupyter notebook containing the main analysis and model development
- `Hate speech detection on twitter data_live.py`: Python script version of the notebook
- `data/`: Directory containing the dataset
  - `train.csv`: Training dataset
  - `test.csv`: Test dataset
